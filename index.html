Manual installation

Add the following link under Settings → Browse → Anime extension repos (Aniyomi) or More → Settings → Browse → Extension Stores (Anime) (Tadami).

…

Same URL works for both apps. Trust the extension after install if prompted.

Usage

Extension sources can be downloaded, installed, and uninstalled via the main app. They are installed and uninstalled like regular apps, in .apk format.

Flow: add repo → Browse → Extensions → install → open Sources → Popular / Latest / Search → play.
